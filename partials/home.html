HTTP REST Requests

Request Description
Get Retrieve single resource or list of resources from a collection.
Query string parameters should allow list filtering.		 GET /datacenters?region=EUROPE&lname=Tes
Post Creates a new resource.
The “Location” HTTP header should return a URL to the newly created resource.
Put Update an existing object.
Delete Delete an existing resource.
Options Returns the “Allow” HTTP header identifying which HTTP requests are permitted against specified resource.
